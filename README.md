# tes123

Este repositorio incluye un ejemplo de mapa mental construido con [Mermaid](https://mermaid.js.org/).

## Cómo usar

1. Abre `mindmap.html` en tu navegador favorito.
2. El diagrama mostrará los nutrientes esenciales de la leche, los tipos por especie y los factores que los afectan.

> Nota: se usa la CDN oficial de Mermaid (v10) para renderizar el mapa mental, por lo que es necesario contar con conexión a internet al abrir el archivo.
